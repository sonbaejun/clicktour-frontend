# ✈ Click Tour



## 🚀 프로젝트 소개
클릭투어는 나만의 여행 플래너를 제작할 수 있는 웹사이트입니다.<br/>여행 플래너를 제작하고 다른 사람들과 공유할 수 있습니다.

## 📝 프로젝트 기능
- 여행 플래너 제작
- 여행 플래너 랜덤 추천
- 플래너 복사하기
- 여행 플래너에 대한 후기 게시판
